# Hashing Techniques in Java

This project demonstrates different hashing collision resolution techniques using Java.

## Features
- Linear Probing
- Quadratic Probing
- Separate Chaining using Linked List
- Insert and Search operations
- Display Hash Tables
- Even/Odd and Divisibility Analysis of IDs
- Menu Driven Program

## Technologies Used
- Java
- Arrays
- Linked Lists
- Hashing

## Collision Resolution Techniques
### 1. Linear Probing
Searches sequentially for the next empty slot.

### 2. Quadratic Probing
Uses quadratic jumps to resolve collisions.

### 3. Chaining
Stores collided elements in linked lists.
